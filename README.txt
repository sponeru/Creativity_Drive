Creativity Drive
================

Creativity Drive is a Minecraft Forge 1.20.1 mod that makes Mekanism Creative
Fluid Tanks and Creative Chemical Tanks usable as infinite AE2 storage cells.
Chemical support is provided through Applied Mekanistics.

Usage
-----

1. Fill a Mekanism Creative Fluid Tank or Creative Chemical Tank with the fluid,
   gas, infusion, pigment, or slurry you want to provide.
2. Put that tank directly into an AE2 ME Drive.
3. AE2 will expose the tank's contents as an effectively infinite source.

The tank does not accept inserted contents through AE2. It only provides the
fluid or chemical already stored in the Mekanism creative tank.

Runtime Dependencies
--------------------

- Mekanism
- Applied Energistics 2
- GuideME
- Applied Mekanistics

Build
-----

Run:

    gradlew.bat build

The built mod jar is written to:

    build/libs/creativity_drive-1.0.0.jar
